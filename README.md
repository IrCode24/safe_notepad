# SafeNotepad
SafeNotepad は、年配の方でも読みやすいように視認性をよくした「シンプルで安全なメモ帳アプリ」です。  

# 開発の目的
　あるシステムに修理履歴を入力する際、もともと定型文をWINDWS標準のメモ帳に入力、保存し、そこからシステムにコピペしていたが
 コピーの際に定型文を誤って削除してしまう事象が発生したため開発に至った。
 また、年配者でPCの使い方で不安を抱える方が多くいたので視認性向上のため文字を大きくできるように

 
# 主な機能
- Python + Tkinter による軽量なデスクトップアプリ
- シンプルメモ編集
- フォントの大きさを直感的に変更できるスライダーを配置
- 視認性向上のためダークモードに切替可能
- 編集可能ボタン(編集可能／不可切替)を配置
- APPDATA に JSON 形式で前回使用したフォントサイズとダークモードを保存


# 技術スタック
- Python 3.14
- Tkinter
- JSON
- Windows（APPDATA 管理）
- VSCode
  

# 技術的な工夫・チャレンジ

- 初心者向け UI/UX  
  余計な機能を排除し、誰でも迷わず使える操作性と視認性を重視。

- 編集モードボタン  
  テキストをコピーする際に誤ってテキストを消さないよう、デフォルトで編集不可としている。
